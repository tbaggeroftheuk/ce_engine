| Level         | Value | Description                      |
| ------------- | ----- | -------------------------------- |
| `LOG_ALL`     | 0     | All logs, very verbose           |
| `LOG_TRACE`   | 1     | Trace-level messages             |
| `LOG_DEBUG`   | 2     | Debug messages                   |
| `LOG_INFO`    | 3     | Informational messages (default) |
| `LOG_WARNING` | 4     | Warnings only                    |
| `LOG_ERROR`   | 5     | Only errors                      |
| `LOG_FATAL`   | 6     | Fatal errors (program crashes)   |
| `LOG_NONE`    | 7     | No logging at all                |
